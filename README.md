# web-screen-recorder

`@devcomfort/web-screen-recorder`는 브라우저의 화면 녹화 기능을 쉽게 사용할 수 있도록 추상화된 녹화 기능을 제공하는 라이브러리입니다.


## Installation

```bash
# npm
npm i @devcomfort/web-screen-recorder
# yarn
yarn add @devcomfort/web-screen-recorder
# pnpm
pnpm i @devcomfort/web-screen-recorder
```
